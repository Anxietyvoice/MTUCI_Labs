# Лабораторные работы по курсу "Введение в информационные технологии"

## Структура репозитория

- `Labs/` - Задания и материалы;
- `Students/` - Работы студентов;
- Repository Instructions - инструкция по работе с репозиторием.

## Лабораторные работы

1. **Лаба 1** - Основные средства и технология разработки консольных программных проектов в интегрированной среде Visual Studio .NET
2. **Лаба 3** - Разработка простейших проектов с использованием основных средств языка программирования VC++
3. **Лаба 4** - Разработка консольных проектов Visual Studio с использованием функций VC++
4. **Лаба 5** - Разработка алгоритмов решения сложных задач методом пошаговой детализации и их программная реализация
5. **Лаба 6-1** - Программная реализация типовых алгоритмов с разветвляющимися структурами
6. **Лаба 7** - Программная реализация типовых алгоритмов с регулярными циклическими структурами
7. **Лаба 8** - Программная реализация типовых алгоритмов с итеративными циклическими структурами

## Правила сдачи

1. Каждый студент выкладывает результаты работы в папку `Students/Группа/Фамилия и Имя/`;
2. Для представления лабораторных работ создаем отдельные подпапки `lab1/`, `lab1_extra/`, `lab3/` и т.д.;
3. В папках `lab1/`, `lab1_extra/`, `lab3/` и т.д. размещаем все файлы решения в соответствии с инструкцией по загрузке материалов;
4. Делаем pull request для проверки.
